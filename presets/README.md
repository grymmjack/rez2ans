0. Go to your git repos dir (e.g. `cd ~/git`)
1. Clone the repo `git repo clone https://github.com/grymmjack/rez2ans`
2. Create a new branch from main: `git checkout -b yourname-presets`
3. Create a new directory with `yourname` inside of `presets`
4. Create a new directory within `presets/yourname/` for: `standard` for standard presets, or `whatever`
5. Copy your `.ini` files into `presets/yourname/standard/`
6. Add your files: `cd ~/git/rez2ans && git add . && git commit -m 'Adding yourname presets'`
7. Push your branch: `git push`
8. Create a Pull Request to merge your branch (`yourname-presets`) into `main` branch
9. If approved, your presets will be added
